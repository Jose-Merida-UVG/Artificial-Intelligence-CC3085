# Hoja de Trabajo 2: Markov Decision Processes

## Summary

This assignment explores the theoretical foundations of Markov Decision Processes and their practical application.

- **The Gambler's Fallacy & the Markov Property**: Mathematical analysis of why including win/loss history violates (or is irrelevant to) the Markov Property in truly random games like Blackjack.

- **Discount Factor Analysis**: Discussion of how the discount factor $\gamma$ shapes agent behavior between myopic and far-sighted strategies.

- **FrozenLake MDP**: Implementation of value iteration on the FrozenLake environment, connecting theoretical MDP concepts to a concrete problem.

## Deliverables

- **Informe.pdf**: Written report with theoretical analysis and discussion.

- **Informe.tex**: Original LaTeX source for the report.

- **FrozenLake.ipynb**: Jupyter notebook with the value iteration implementation.
