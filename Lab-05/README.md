# Lab 5: Reinforcement Learning

## Summary

This assignment explores reinforcement learning theory and practice using OpenAI Gym's FrozenLake environment.

- **RL Theory**: Analysis of the exploration-exploitation dilemma, reward shaping, and policy design considerations through engineering-oriented discussion.

- **FrozenLake**: Implementation of reinforcement learning algorithms to solve the FrozenLake environment, including value-based methods and policy evaluation.

- **Experiment Results**: Recorded training runs and performance metrics stored in the `gym-results/` directory.

## Deliverables

- **Informe.pdf**: Written report with theoretical analysis, experiment results, and discussion.

- **Informe.tex**: Original LaTeX source for the report.

- **FrozenLake.ipynb**: Jupyter notebook with the RL implementation and experiments.

- **gym-results/**: Directory containing recorded environment runs and training metrics.

## Execution

Before running the notebook, make sure [Python](https://www.python.org) is installed.

```bash
pip install -r requirements.txt
jupyter notebook FrozenLake.ipynb
```
