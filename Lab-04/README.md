# Lab 4: SGD & Neural Networks

## Summary

This assignment dives into optimization with Stochastic Gradient Descent and introduces neural network fundamentals through the Perceptron model.

- **Stochastic Gradient Descent**: Implementation and analysis of SGD, including learning rate sensitivity, convergence behavior, and comparison against batch gradient descent.

- **Perceptron**: Construction of a single-layer Perceptron, exploring its capabilities and limitations as a linear classifier, and discussion of activation functions and network design.

- **Theoretical Analysis**: Engineering-oriented discussion of optimization landscapes, neural architecture design decisions, and their practical consequences.

## Deliverables

- **Informe.pdf**: Written report with theoretical analysis, experiment results, and discussion.

- **Informe.tex**: Original LaTeX source for the report.

- **SGD.ipynb**: Jupyter notebook with the SGD implementation and experiments.

- **Perceptron.ipynb**: Jupyter notebook with the Perceptron implementation.

## Execution

Before running the notebooks, make sure [Python](https://www.python.org) is installed.

```bash
pip install -r requirements.txt
jupyter notebook SGD.ipynb
jupyter notebook Perceptron.ipynb
```
