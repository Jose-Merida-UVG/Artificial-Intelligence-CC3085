# Lab 2: KNN & Bias-Variance

## Summary

This assignment covers data preparation fundamentals, the convexity of logistic regression, and KNN classification with a focus on the bias-variance tradeoff.

- **Logistic Regression Convexity**: Theoretical analysis of why the cross-entropy loss function is convex and its implications for optimization guarantees.

- **KNN & Bias-Variance Tradeoff**: Implementation and experimentation with K-Nearest Neighbors, exploring how the choice of $k$ affects model complexity, underfitting, and overfitting.

- **Data Preparation**: Preprocessing pipelines including feature scaling, encoding, and train-test splitting applied before model training.

## Deliverables

- **Informe.pdf**: Written report with theoretical analysis, experiment results, and discussion.

- **Informe.tex**: Original LaTeX source for the report.

- **Notebook.ipynb**: Jupyter notebook containing the KNN implementation, data preparation, and bias-variance experiments.
