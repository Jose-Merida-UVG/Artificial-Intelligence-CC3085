## About This Repository

This repository hosts the academic assignments and projects for the **Artificial Intelligence (CC3085)** course at Universidad del Valle de Guatemala. Each subfolder corresponds to a different assignment, detailed in the table of contents below. Inside each subfolder you'll find a dedicated `README.md` with assignment details, alongside written reports and project-specific documentation authored in LaTeX.

## Structure

### Labs

| Assignment | Topic | Link | Description |
| :--------- | :---- | :--- | :---------- |
| **Lab-01** | Performance Metrics & Data Cleaning | [View Folder](./Lab-01/) | Business case analysis (Precision/Recall & adjusted $R^2$), manual data cleaning (imputation & undersampling), and MAE/RMSE implementations with $L_p$ norm sensitivity analysis. |
| **Lab-02** | KNN & Bias-Variance | [View Folder](./Lab-02/) | Logistic regression convexity analysis, KNN implementation with bias-variance tradeoff exploration, and data preparation pipelines. |
| **Lab-03** | Naive Bayes, SVM & Decision Trees | [View Folder](./Lab-03/) | Spam filtering with Naive Bayes, sports league classification with SVMs, and decision tree analysis with theoretical discussion on independence assumptions. |
| **Lab-04** | SGD & Neural Networks | [View Folder](./Lab-04/) | Stochastic Gradient Descent implementation, Perceptron model construction, and theoretical analysis of optimization and neural network design decisions. |
| **Lab-05** | Reinforcement Learning | [View Folder](./Lab-05/) | FrozenLake environment solved with reinforcement learning algorithms, including theoretical analysis of MDPs and policy design. |

### Hojas de Trabajo

| Assignment | Topic | Link | Description |
| :--------- | :---- | :--- | :---------- |
| **HT-1** | Agents & Environments | [View Folder](./HT-1/) | PEAS analysis for a solar cleaning robot, metric design for a traffic system in Guatemala City, and a simple reflex agent implementation. |
| **HT-2** | Markov Decision Processes | [View Folder](./HT-2/) | Theoretical analysis of MDPs, the Gambler's Fallacy vs. the Markov Property, discount factor effects, and a FrozenLake value iteration implementation. |
