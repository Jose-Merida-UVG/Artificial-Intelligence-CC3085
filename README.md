## About This Repository

This repository hosts the academic assignments and projects for the **Artificial Intelligence (CC3085)** course at Universidad del Valle de Guatemala. Each subfolder corresponds to a different assignment, detailed in the table of contents below. Inside each subfolder you'll find a dedicated `README.md` with assignment details, alongside written reports and project-specific documentation authored in LaTeX.

## Structure

### Labs

| Assignment | Topic | Link | Description |
| :--------- | :---- | :--- | :---------- |
| **Lab-01** | Performance Metrics & Data Cleaning | [View Folder](./Lab-01/) | Business case analysis (Precision/Recall & Adjusted $R^2$), manual data cleaning pipeline (imputation & undersampling), and from-scratch MAE/RMSE with $L_p$ norm sensitivity analysis. |
| **Lab-02** | Data Preparation, KNN & Bias-Variance | [View Folder](./Lab-02/) | Logistic regression convexity analysis, KNN and Logistic Regression implemented from scratch and benchmarked against scikit-learn on a phishing detection dataset. |
| **Lab-03** | Naive Bayes, SVM & Decision Trees | [View Folder](./Lab-03/) | From-scratch Bayesian spam filter on real SMS data, SVM and Decision Tree classifiers applied to League of Legends match data, with theoretical discussion on independence assumptions and greedy myopia. |
| **Lab-04** | SGD & Neural Networks | [View Folder](./Lab-04/) | From-scratch Batch GD, SGD, and Mini-Batch GD on a noisy polynomial, Perceptron classifier on Iris, and theoretical analysis of vanishing gradients and depth vs. width. |
| **Lab-05** | Reinforcement Learning | [View Folder](./Lab-05/) | Q-Learning on the stochastic FrozenLake-v1 environment (Gymnasium), with theoretical analysis of exploration/exploitation, SARSA vs. Q-Learning, and bootstrapping. |
| **Lab-06** | Minimax & Adversarial Search | [View Folder](./Lab-06/) | Connect Four agent using Minimax with Alpha-Beta pruning (93.86% node reduction), custom heuristic evaluation, and an interactive Streamlit web app. |
| **Lab-08** | Constraint Satisfaction Problems | [View Folder](./Lab-08/) | Microservice-to-server allocation modeled as a CSP, solved with Backtracking, MRV + Forward Checking, Beam Search, and ICM local search, with comparative analysis. |

### Hojas de Trabajo

| Assignment | Topic | Link | Description |
| :--------- | :---- | :--- | :---------- |
| **HT-1** | Agents & Environments | [View Folder](./HT-1/) | Composite performance metric design for a Guatemala City traffic system, full PEAS analysis for an autonomous solar panel cleaning robot in Zacapa, and a Simple Reflex Agent in Python. |
| **HT-2** | Markov Decision Processes | [View Folder](./HT-2/) | Theoretical analysis of the Markov Property, discount factor behavior, and living penalties, plus a from-scratch Value Iteration implementation on FrozenLake with policy visualization. |
